# 更新记录

## v0.4.5 - 2026-07-05

- 新增科研通每日打卡自动检测与状态显示。
- 扩展下载通道识别，支持 `View PDF`、`Download PDF` 以及 Springer 等出版商 PDF 入口。
- 新增多类不纳入应助范围的自动拦截：Semantic Scholar、Cell / Cell Press、求助补充材料、已完成、已有人上传等待确认。
- 拦截后自动返回科研通 `求助中 + Elsevier` 列表，方便继续处理下一条。
- 保留真实 PDF 校验与下载路径复制，避免把 HTML 页面误当作 PDF。

## v0.4.3 - 2026-07-03

- 插件正式命名为“一键应助”。
- 增加插件图标与基础 GitHub 文档。
- 支持一键应助流程、求助列表、复制最新 PDF 路径、复制文献标题。
- 支持 ScienceDirect、Springer、JPD 等页面的 PDF 流程基础处理。
